# Solana Editions

# Install Anchor 

### AVM

```bash
# repo root
cargo install --git https://github.com/project-serum/anchor avm --locked --force

avm install latest
avm use latest
```

### Verify Anchor Is Working

```bash
# repo root
anchor --version
```


# Solita

```bash
# repo root
yarn add -D @metaplex-foundation/solita
yarn solita # automatically generates typescript bindings
```
